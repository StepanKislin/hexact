# ℹ️ О проекте HexAct

## 🎯 Миссия проекта

**HexAct** создан для автоматизации процесса контроля качества крепежных изделий на производственных линиях и в складских помещениях. Проект демонстрирует возможности современных веб-технологий в области компьютерного зрения и промышленной автоматизации.

## 🎓 Технологические вызовы

### Реализованные решения

#### 1. Детекция геометрических форм
- **Проблема:** Точное распознавание различных типов головок
- **Решение:** Комбинация контурного анализа и аппроксимации полигонов
- **Результат:** 92.1% точности на тестовых данных

#### 2. Отслеживание объектов
- **Проблема:** Устранение дубликатов при детекции одного объекта в нескольких кадрах
- **Решение:** Центроидное отслеживание с временным окном
- **Результат:** Стабильная работа при 14.8 FPS

#### 3. Распознавание крестообразных шлицев
- **Проблема:** Обнаружение перпендикулярных линий внутри окружности
- **Решение:** Комбинация преобразования Хафа и анализа углов
- **Результат:** 95% точности для болтов Phillips

#### 4. Работа с фоном
- **Проблема:** Разделение объектов от фона при различных условиях освещения
- **Решение:** Калибровка фона и дифференциальный анализ
- **Результат:** Улучшение детекции на 30%

## 📊 Метрики качества

### Точность классификации
- **Шестигранник (болт):** 95%
- **Крест (Phillips):** 95%
- **Шестигранник (гайка):** 95%
- **Пятиугольник:** 80%
- **Семиугольник:** 80%
- **Восьмиугольник:** 80%
- **Незавершенный шестигранник:** 75%
- **Круг без креста:** 80%

### Производительность
- **Средняя частота кадров:** 14.8 FPS
- **Время обработки кадра:** ~67 мс
- **Использование памяти:** ~50-100 МБ
- **Загрузка CPU:** 30-50% (на среднем ПК)

### Выбор метрик
При значительном дисбалансе классов наиболее подходящими метриками являются:
- **F1-Score** — гармоническое среднее точности и полноты
- **AUC-ROC** — площадь под кривой ошибок
